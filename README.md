# RAD Editor for Kids

This is a RAD Editor for kids, no login required, everything is immutable

## Goals

The goal of this project is to create a full web/mobile experience environment
that kids can create high quality websites, web apps and responsive mobile apps.

## Development setup

* install nodejs - (https://nodejs.org)
* checkout repo - `git clone ...`
* `cd rad`
* `npm install`
* `npm start`

## Features

* Ability to drag and drop using blockly to create web driven applications
* Ability to save and retrieve the project using the url specified
* Ability to publish project either as a module or application
* Ability to use published modules in your module and application
